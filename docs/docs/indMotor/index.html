Induction Motor

An asynchronous machine of which only one winding is energized. IEC 60050.

Three-phase induction motor in PSP-UFU​

In PSP-UFU, induction motors are included in load flow and stability studies.

Information

The stability data of the induction machine are used together with the load flow, correctly calculating the reactive power of the machines and consequently the voltage at the connected bus.

Warning!

When selecting the option "Calculate reactive power in load flow", the correct data must be entered in the "Stability" tab, otherwise an incorrect reactive power will be assigned to the motor.

As seen in the above equations, there are four variables and only two equations. In practice, the variables can be reduced to three since the voltage magnitude (V) is obtained from the power flow results.
To solve the equations, one more variable must be defined. In PSP-UFU, the chosen fixed variable is active power (P), as it provides numerically correct and suitable results for motors under stability conditions.

This model can be inserted into the load flow solution by following these steps:

  1. Calculate constants K1K_1K1​ to K4K_4K4​. These values remain constant throughout the solution;
  2. In each iteration, calculate coefficients AAA, BBB, and CCC using the updated value of VVV;
  3. Solve the quadratic equation to obtain two values of (r2s)\left(\frac{r_2}{s} \right)(sr2​​); choose the larger one as it is in the stable region of the torque-slip characteristic;
  4. Using the new (r2s)\left(\frac{r_2}{s} \right)(sr2​​) value, obtain the reactive power (QQQ). The power vector is updated and conventional load flow solution steps are executed.

Steps 2 to 4 are repeated until convergence is achieved.

Attention!

In PSP-UFU, induction motors are not considered in short-circuit calculations.

Three-phase induction motor in stability study​

An important dynamic load is the induction motor since it accounts for a significant share of system loads.
The induction machine model presented earlier, which can operate as both motor and generator, is well established in the literature.

As described in the previous section, initialization of this machine is done together with the power flow since the reactive power required by the induction machine depends on motor parameters as well as bus voltage. This approach is necessary because conventional methods lead to incorrect results in heavily loaded systems.

The induction machine motion equation must be expressed in terms of torque, not power, as done with synchronous machines. Rotor symmetry means angular position is not important, and slip (sss) is used instead of speed (ω\omegaω), where:

Induction motor data editing form​

The figure below shows the data insertion/editing form for induction motors:

The form includes the "Switching" button in the lower-left corner.
This form, common to several other elements, allows the insertion and/or removal of the motor during the stability study.

  • General
  • Stability
  • Switching Button

Name​

Electrical element identification. Any number of characters can be entered using the Unicode standard.

All PSP-UFU power components have this field.

Rated power​

Motor rated power, entered in MVA, kVA, or VA.

This field is especially important if the "Use rated power as base" option is checked.

Active and reactive powers​

Motor active power (entered in W, kW, MW, or p.u.p.u.p.u.) and reactive power (entered in var, kvar, Mvar, or p.u.p.u.p.u.).

Attention!

If the “Calculate reactive power in load flow” option is enabled, the reactive power field is disabled for editing.

Calculate reactive power in load flow​

If selected, the program uses the stability form data to calculate the motor’s reactive power during the iterative load flow process.

Warning!

If this option is not used, the motor will be considered a constant power load in the load flow study.

Not using this option may cause steady-state errors in the stability study.

Use rated power as base​

If selected, the program uses the motor’s rated power as the base for unit conversion, including those in the stability form; otherwise, the system base power is used.

Print induction machine data​

Displays induction motor data in time-domain plots. The following data are shown:

  • terminal voltage
  • active and reactive powers
  • current, electrical torque
  • mechanical torque
  • speed
  • slip

Inertia​

Inertia of the motor-load set, in seconds.

Stator resistance and reactance​

Stator resistance and reactance of the induction motor, in p.u.p.u.p.u.

Rotor resistance and reactance​

Rotor resistance and reactance of the induction motor, in p.u.p.u.p.u.

Magnetizing reactance​

Magnetizing branch reactance of the induction motor, in p.u.p.u.p.u.

Use cage factor​

Allows inserting a cage factor (KgK_gKg​) in the induction motor.
This factor increases rotor resistance under high slip conditions, as in double-cage or deep-bar motors. Rotor resistance follows:

r2=r20(1+Kgs)r_2 = r_{2_0} \left( 1+K_g s \right)r2​=r20​​(1+Kg​s)

Where:

  • r20r_{2_0}r20​​ = rotor resistance entered in the form.

Load characteristic​

Describes the mechanical load torque characteristic on the motor shaft according to the mechanical torque equation:

Tm∝a+bω+cω2T_m \propto \bold{a} + \bold{b}\omega + \bold{c}\omega^2Tm​∝a+bω+cω2
Warning!

The sum of the coefficients (a+b+ca + b + ca+b+c) must be equal to one.

The "Switching" button opens a form, common to several other elements, allowing insertion and/or removal of the motor during the stability study.

In this form, a generic list of insertions and removals over time can be created, customized by a switching properties context where the switching type (insertion or removal) and event time (in seconds) are edited. These properties are added or removed from the list using the "Add" and "Remove" buttons, respectively.
