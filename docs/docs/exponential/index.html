Exponencial

A função exponencial dos elementos de controle realiza essa operação com uma entrada real, obedecendo a expressão:

yn=AeBuny_n = \bold{A} e^{\bold{B} u_n}yn​=AeBun​

Em que:

  • unu_nun​ e yny_nyn​ é a entrada e a saída do bloco exponencial, respectivamente;
  • A\bold{A}A e B\bold{B}B são constantes definidas pelo usuário.
